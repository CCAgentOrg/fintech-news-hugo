# 💰 Cashless Watch

An AI-powered, fully automated daily newsletter tracking India's fintech ecosystem — UPI, RBI regulations, startup funding, banking, and policy. Content is generated by autonomous agents and published to GitHub Pages.

## 🌐 Live Site

**URL**: https://newsletter.cashlessconsumer.in

## 🤖 How It Works

This blog is entirely automated by two AI agents:

| Agent | Schedule | Coverage | Source |
|-------|----------|----------|--------|
| **Daily Fintech Brief** | Daily @ 8:00 AM IST | Last 24 hours | RBI, SEBI, NPCI, startup funding |
| **Themed Deep Dive** | Daily @ 8:30 AM IST | Day-specific analysis | Global fintech, Open Banking, Policy |

### Themed Rotation

| Day | Theme | Focus Areas |
|-----|-------|-------------|
| **Mon** | Developer & APIs | Open Banking, bank APIs, RBI tech stack, developer tools |
| **Tue** | Buzz & Trends | News cycles, Twitter/X sentiment, funding news |
| **Wed** | Global View | Cross-border payments, international fintech trends, India Stack exports |
| **Thu** | Regulatory | RBI circulars, SEBI notifications, policy updates |
| **Fri** | Payments | UPI, NPCI, wallets, cards, real-time payments |
| **Sat** | Deep Dive | Weekly fintech roundup, analysis synthesis |
| **Sun** | Policy Newsletter | Finance Ministry, GST, tax, economic policy |

**Model**: minimax-m2.5  
**Language**: en-IN (Indian English)  
**Sources**: 15+ oracle authorities

## 🛠️ Local Development

```bash
# Clone
git clone https://github.com/CCAgentOrg/fintech-news-hugo.git
cd fintech-news-hugo

# Install Hugo (extended version)
# https://gohugo.io/installation/

# Init theme submodule
git submodule update --init --recursive

# Local preview
hugo server -D

# Build
hugo --gc --minify
```
